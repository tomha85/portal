# Safety Vision Portal

Production-style React + TypeScript enterprise dashboard UI for industrial PPE and safety analytics.

## Stack
- React + TypeScript + Vite
- Tailwind CSS
- shadcn-style UI primitives
- lucide-react icons
- Recharts
- Framer Motion

## Run
```bash
npm install
npm run dev
```

## Routes
Overview, Live Monitoring, Alerts, Incidents, Analytics, Facilities, Cameras, Rules & Zones, Reports, Admin.

All data is mocked in `src/data/mockData.ts` and strongly typed for easy backend integration.
